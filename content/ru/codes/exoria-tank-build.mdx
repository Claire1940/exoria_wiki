## Сборка аквариума (Fish Tank): Генерация ресурсов

В Exoria аквариум (Fish Tank) — это не просто декоративный элемент; это функциональная машина, используемая для разведения водных обитателей, которые дают необходимые материалы. Создание правильной установки аквариумов — один из первых крупных проектов типа «tank», за которые вы возьметесь. Эти резервуары позволяют разводить такие виды, как иглобрюх (Pufferfish), которые используются в различных алхимических рецептах и рецептах инфузии.

Для начала вам нужно будет скрафтить аквариумы, используя стеклянные панели и ведра с водой. Поскольку эти блоки функционируют как камеры для разведения, их часто приходится строить в больших количествах для максимизации производительности.

| Компонент | Количество (рекомендуемое) | Назначение |
| :--- | :--- | :--- |
| **Блок аквариума (Fish Tank)** | 32 - 64 | Основной блок содержания и разведения. |
| **Ведра с водой** | 1 на аквариум | Требуется для инициации водной среды. |
| **Семена** | Варьируется | Используются как приманка/корм для запуска процесса разведения. |
| **Сырая рыба** | 2+ на вид | Стартовая популяция для вашей фермы. |

> **💡 Совет:** При настройке рыбной фермы размещайте аквариумы в конфигурации 2x2 или больше. Это позволяет рыбам двигаться более свободно и облегчает взаимодействие с ними для разведения.

## Питание вашей инфраструктуры: Двигатели Стирлинга и BuildCraft

Надежный **exoria tank build** требует стабильной подачи энергии для работы продвинутых механизмов, таких как сборочный стол (Assembly Table) и лазеры. В середине игры вы, скорее всего, перейдете на энергию BuildCraft (MJ), чтобы облегчить создание редстоун-чипсетов. Эти чипсеты являются основой более сложного «прочного» оборудования.

Двигатель Стирлинга — самый надежный источник энергии начального уровня. Он принимает любое топливо, подходящее для обычной печи, например, древесный или каменный уголь. Однако управление уровнем нагрева имеет решающее значение для предотвращения поломки двигателя.

### Требования к установке двигателя
Для эффективной работы сборочного стола с пятью лазерами вам следует организовать отдельное энергетическое помещение. Оно включает в себя:
1.  **Двигатели Стирлинга:** Минимум 5–6 двигателей для поддержания стабильного потока MJ.
2.  **Кинетические трубы:** Используйте деревянные кинетические трубы на выходе двигателя и булыжниковые кинетические трубы для основной линии.
3.  **Управление редстоуном:** Используйте рычаг и редстоун-пыль для включения и выключения массива двигателей, чтобы предотвратить перегрев, когда они не используются.

| Тип трубы | Макс. поток (прибл.) | Применение |
| :--- | :--- | :--- |
| **Деревянная кинетическая труба** | 16 MJ/t | Извлекает энергию напрямую из двигателя. |
| **Булыжниковая кинетическая труба** | Без ограничений (мод.) | Транспортирует энергию к сборочному столу. |
| **Кварцевая кинетическая труба** | Высокий | Лучше всего подходит для передачи энергии на большие расстояния с низкими потерями. |

## Продвинутое выживание: Инфузор жизни (Life Infuser)

Выживание в глубинах Exoria требует предметов, которые нельзя просто скрафтить на верстаке. Инфузор жизни — это мультиблочная структура из мода Modular Machines, которая позволяет «наполнять жизнью» обычные объекты. Именно так вы получаете такие предметы, как ведро с треской или специализированное снаряжение для выживания.

Для постройки Инфузора жизни требуется определенная планировка из рунного камня (Rune Stone), контроллеров машин и крошечных входных/выходных портов.

### Постройка структуры Инфузора жизни
*   **Контроллер машины:** «Мозг» структуры, крафтится из медных слитков.
*   **Крошечные входы для предметов:** Вам понадобятся три таких входа для подачи сырья в машину.
*   **Крошечный выход для предметов:** Одного достаточно для сбора готовой продукции.
*   **Кирпичи из рунного камня:** Используются для колонн и обеспечения структурной целостности.

> **⚠️ Предупреждение:** Убедитесь, что структура построена в точном соответствии с чертежом. Используйте функцию «Preview» (Предпросмотр) на контроллере, чтобы увидеть призрачное изображение того, где должны быть размещены блоки. Отсутствие хотя бы одного блока не позволит машине сформироваться.

## Обработка материалов для «танковой» защиты

Чтобы по-настоящему достичь статуса «танка» в Exoria, вы должны перерабатывать сырую руду в высокотехнологичные сплавы. Металлический пресс (Metal Press) и штамповщик (Stamper) — ваши основные инструменты. Например, создание пластин из рассветного камня (Dawnstone Plates) необходимо для крафта «прочных корпусов» (Sturdy Casings), которые используются почти в каждой продвинутой лесной или промышленной машине.

1.  **Расплавьте руду:** Используйте плавильню на энергии Ember, чтобы превратить рассветный камень или медь в жидкую форму.
2.  **Извлеките жидкость:** Используйте экстракторы жидкости и трубы, чтобы переместить жидкость в штамповщик.
3.  **Отштампуйте пластину:** Поместите форму для пластин в штамповщик, чтобы произвести необходимые компоненты для вашего **exoria tank build**.

| Машина | Вход | Выход | Источник энергии |
| :--- | :--- | :--- | :--- |
| **Штамповщик (Stamper)** | Жидкий металл | Пластины/Шестерни | Ember |
| **Сборочный стол** | Редстоун/Железо | Чипсеты | MJ (BuildCraft) |
| **Соковыжималка (Squeezer)** | Семена/Фрукты | Семенное масло/Сок | RF или MJ |

## Спуск в глубины: Портал Exoria

Как только ваши ресурсные резервуары и энергетические системы будут налажены, финальным этапом вашей ранней сборки для выживания станет уход с поверхности. Ресурсы на поверхности планеты со временем заканчиваются, что вынуждает вас исследовать пещеры.

Для этого вы должны создать **Артефакт Exoria**. Для него требуется изумруд (получаемый через сито с алмазной сеткой) и различные кристаллы, разбросанные по миру. Когда артефакт будет готов, постройте раму портала из рунного камня. Этот портал — ваши ворота на следующий уровень игры, где окружающая среда значительно более враждебна, что делает вашу предыдущую работу над **exoria tank build** и броней еще более важной.

Вы можете найти более подробную техническую документацию на [официальной странице Exoria на CurseForge](https://www.curseforge.com/minecraft/modpacks/exoria), которая поможет разобраться в специфических взаимодействиях модов.

## Итоги этапов выживания
1.  **Автоматизируйте переработку пепла:** Используйте редстоуновые двигатели и сундуки для сбора материалов для алмазной сетки.
2.  **Постройте аквариум:** Обеспечьте возобновляемый источник рыбы для рецептов Инфузора жизни.
3.  **Наладьте питание BuildCraft:** Установите двигатели Стирлинга для создания редстоун-чипсетов.
4.  **Сконструируйте Инфузор жизни:** Используйте компоненты модульных машин для создания предметов уровня выживания.
5.  **Активируйте портал:** Используйте Артефакт Exoria на раме из рунного камня, чтобы спуститься в пещеры.

## FAQ

**В: Как получить изумруды для Артефакта Exoria?**

О: Вы должны просеивать пыль или пепел через сито с **алмазной сеткой (Diamond Mesh)**. Шанс выпадения составляет примерно 10%, поэтому вам нужно будет автоматизировать сбор пепла, чтобы получить достаточно материалов.

**В: Мои двигатели Стирлинга краснеют. Они взорвутся?**

О: Да, во многих версиях BuildCraft двигатели Стирлинга могут перегреваться и в конечном итоге взрываться, если их не охлаждать или не выключать. Всегда используйте рычаг, чтобы отключать энергосеть вашего **exoria tank build**, когда сборочный стол простаивает.

**В: Как лучше всего разводить рыбу в аквариуме?**

О: Поместите как минимум две рыбы одного вида в аквариум и щелкните по ним правой кнопкой мыши, держа в руке семена. Это запустит механику разведения, аналогичную разведению животных в обычном Minecraft.

**В: Почему мой Инфузор жизни не работает?**

О: Проверьте наличие всех трех крошечных входов для предметов и контроллера машины. Также убедитесь, что вы используете правильный «Камень инфузии» (например, песчаниковый камень инфузии) внутри интерфейса машины.
